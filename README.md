## Getting Started

### Installation

1. Clone the repository

2. Install dependencies:

```bash
npm install
```

3. Start development server:

```bash
npm run dev
```

### Building for Production

```bash
npm run build
```

The built files will be generated in the `dist` directory.

### Running Production Build

```bash
npm run start
```
